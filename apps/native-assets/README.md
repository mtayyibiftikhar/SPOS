# Native App Assets

This folder is the source home for Windows and Android app branding.

- `app-icon.svg` is the master icon artwork.
- Windows installer icons can be generated from this artwork as `app-icon.ico`.
- Android launcher icons and splash assets can be generated from this artwork inside the Capacitor Android project.

Changing shop logos inside the POS is still done from `Settings > Shop Settings`.
Changing POS company branding is still done from the owner branding area.
